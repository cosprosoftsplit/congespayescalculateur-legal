Questions fréquentes

Comment calculer mes congés payés ?

Entrez votre date d'embauche et vos périodes travaillées. L'application calculera automatiquement le nombre de jours de congés payés auxquels vous avez droit.

L'application respecte-t-elle le Code du travail français ?

Oui. Le calcul est basé sur les règles du Code du travail français, incluant les 2,5 jours ouvrés par mois travaillé.

Quelle est la différence entre jours ouvrables et ouvrés ?

L'application prend en charge les deux modes de calcul : en jours ouvrables (lundi à samedi) et en jours ouvrés (lundi à vendredi).

L'application ne fonctionne pas correctement. Que dois-je faire ?

Essayez les étapes suivantes :

  • Assurez-vous d'avoir la dernière version de l'Application installée
  • Redémarrez l'Application
  • Redémarrez votre appareil
  • Réinstallez l'Application

Si le problème persiste, veuillez nous contacter à ivan@cosmicproduction.hr avec une description du problème.

Comment supprimer mes données ?

Toutes les données étant stockées localement sur votre appareil, vous pouvez les supprimer dans l'Application ou en désinstallant l'Application.
